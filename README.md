# YaDev

YaDev is a website that connects Youtubers and Game developers, with a focus on connecting the lesser exposed channels / games with eachother.

## Visiting

YaDev can be found at `https://yadev.app`. 

## Running

* Clone this repository
* CD into directory
* Run `yarn start`
